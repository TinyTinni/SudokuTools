# Sudoku Tools
Some tools for Sudoku handling.
The tools are standalone and can be combined, the file format is a simple 9x9 csv file, where the unknown entries can be 0 or empty.

Current tools:

- [Solver](./solver/)
- [svg Generator](./svg_gen/)
